# Bubbly Heritage Asset Generation Prompts - V2

These prompts are designed to create a consistent, high-end "bubbly" aesthetic for the Congkak RPG. The style is glossy, vibrant, and slightly 3D with thick soft outlines.

## Core Aesthetic: "Bubbly Heritage"
- **Style**: Glossy 3D modeling, rounded edges, vibrant saturation.
- **Lighting**: Soft studio lighting with clear specular highlights (glassy look).
- **Background**: Isolated on transparent background or simple flat neutral color (for easy removal).
- **Colors**: Rich golds, vibrant teals, warm terra cottas, and pearlescent whites.

---

## Instructions for Integration
1. Generate the images using a high-quality AI generator.
2. Remove backgrounds (convert to PNG).
3. Place in `public/assets/textures/ui/icons/`.
4. Update `style.css` to use these icons in the `.top-nav-icon` containers.
